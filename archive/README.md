---

## Why Archive?

We maintain this archive for:

1. **Historical reference** - See how the guide evolved
2. **Attribution** - Credit original sources
3. **Recovery** - Original content available if needed
4. **Transparency** - Show our sources and process

## Note on Linting

`.prettierignore` and `.markdownlint-cli2.jsonc` for configuration.
