## ğŸ¯ How Dork Packs Are Organized

Each pack follows a consistent structure:

### Quick Start
Single most useful query - works immediately, no customization needed.

### Basic Queries
2-3 simple queries with explanations. Good starting points for each sub-topic.

### Intermediate Queries
More precise queries with operator combinations. For users comfortable with core operators.

### Advanced Queries
Complex multi-operator queries for power users and edge cases.

- Links to relevant [Synonym Blocks](../05-synonym-blocks.md)

## ğŸ’¡ Tips for Using Dork Packs

1. **Start with Quick Start** - Get results immediately
2. **Customize the domain** - Replace `site:*.gov.au` with your specific state if needed
3. **Update dates** - Replace `after:2024` with current dates
4. **Combine packs** - Many searches benefit from combining terms from multiple packs
5. **Check Synonym Blocks** - Expand your searches with pre-built term groups
